# trading-simulator
Created a Claude and Yahoo Finance API-powered stock market simulator with live, accurate data from US stock markets to provide beginner traders with a platform to practice trading and research stocks without risking hard-earned money.

# tools used
- Github to host the files, VS Code for coding
- Claude API to run research on companies to provide recent news, trends, and information to allow traders to make informed decisions on engaging with those stocks
- Claude and Github Copilot to assist with coding the project (my first venture into vibe coding!)
